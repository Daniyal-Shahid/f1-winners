# F1 Winners Prediction App

A machine learning-powered application for predicting Formula 1 race outcomes.

## Project Structure

hi

## Features

- Race outcome predictions based on historical data
- Driver performance analysis
- Real-time data updates during race weekends
- Prediction accuracy tracking
- Detailed insights for predictions

## Tech Stack

### Frontend
- React with Vite
- TypeScript
- TailwindCSS
- Chart.js for visualization

### Backend
- Node.js
- Express
- PostgreSQL
- Redis for caching

### Predictions Service
- Python/Flask
- FastF1
- OpenF1
- Scikit-learn
- Pandas

## Getting Started

1. Clone the repository
2. Install dependencies for each service
3. Set up environment variables
4. Run development servers
